# Herring × eBird Version 2 — Comprehensive Analysis Plan

The self-contained reader report is [`reports/comprehensive_analysis_plan.html`](../reports/comprehensive_analysis_plan.html).

This plan contains:

- 50 registered analysis modules spanning counts, occurrence, timing, distance, redistribution, guilds, community composition, co-occurrence, herring measurement, phenology, observation process, validation, and synthesis;
- 18 illustrative synthetic figures showing the intended presentation of results;
- lay summaries paired with technical specifications;
- an explicit distinction among encounter probability, positive reported count, total count, spatial allocation, richness, composition, and residual species association;
- a phased implementation and validation roadmap;
- privacy and provenance requirements for copying reusable Version 1 assets into a clean Version 2 repository.

All figures in the HTML are synthetic examples generated with seed `1988`. They are not Version 2 results.

The machine-readable module registry is [`metadata/analysis_module_registry.csv`](../metadata/analysis_module_registry.csv).
