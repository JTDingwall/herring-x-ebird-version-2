# Repository agent rules

- Treat raw EBD/SED and record-level derivatives as restricted.
- Never commit source checklist, observer, locality, or exact coordinate identifiers.
- Preserve raw inputs and checksums.
- Declare every join cardinality and test it.
- Keep detection, numeric count, `X`, lower-bound count, and ambiguity distinct.
- Missing herring components are not zero; relative spawn index is not absolute biomass.
- Use all concurrent event links for additive exposure; do not silently duplicate checklists as independent rows.
- All prespecified models are reported regardless of sign.
- Hard stops and warnings follow `docs/04_DECISION_RULES.md`.
- Current data analyses are exploratory/estimand-refining until prospective confirmation.
