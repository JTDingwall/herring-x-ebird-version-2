# Optional blinded-companion anonymization log

The journal uses single-anonymized review, so the unblinded manuscript is the correct submission file. The optional blinded companion makes only these substitutions:

1. Author name and ORCID → `[Blinded for review]`.
2. Affiliation, email, postal address, and telephone fields → generic blinded note.
3. Repository URL and exact freeze identifier → withheld-review wording.
4. CRediT, funding, competing-interest, acknowledgments, and AI declaration → blinded placeholders.
5. Supplement author identity → `[Blinded for review]`.

Scientific methods, aggregate results, citations, figures, tables, and interpretation are unchanged. No scientific detail was removed for anonymization.
